# Student Performance Analysis Project

## Project Overview
This project analyzes student performance using statistical concepts, probability theory, and basic linear algebra. The analysis is performed using Python with libraries such as NumPy, Pandas, Matplotlib, Seaborn, and SciPy.

The dataset includes student academic records such as:
- Student_ID
- Age
- Math_Score
- Science_Score
- English_Score
- Hours_Studied
- Pass_Fail

## Objectives

- Calculate measures of central tendency (Mean, Median, Mode)
- Analyze dispersion (Variance, Standard Deviation)
- Compute probability of passing
- Evaluate conditional probability
- Analyze score distribution using histograms and Q-Q plots
- Measure skewness and kurtosis
- Apply linear algebra concepts (Dot Product, Norm, Angle Between Vectors)

## Technologies Used

- Python 3.x
- NumPy
- Pandas
- Matplotlib
- Seaborn
- SciPy

## Key Analysis Performed

### Statistical Analysis
- Mean, Median, Mode
- Variance & Standard Deviation
- Distribution visualization

### Probability Analysis
- Overall passing probability
- Conditional probability based on study hours

### Distribution Testing
- Histogram with KDE
- Q-Q Plot
- Skewness & Kurtosis

### Linear Algebra
- Vector representation of subject scores
- Dot product calculation
- Norm (magnitude) of vectors
- Angle between subject performance vectors

## Project Insights

- Most student scores are centered around the mean.
- Study hours significantly impact passing probability.
- Score distributions approximately follow a normal distribution.
- Statistical and linear algebra techniques help explain performance trends.
